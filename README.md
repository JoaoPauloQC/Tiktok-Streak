# Este é um código simples que mantém a sequência do tiktok ( conhecido como foguinho )

## Instruções: caso queira usar em seu computador, adicione as imagens ( capturas de tela ) que contenham o nome de usuário do tiktok de quem você quer manter a sequência na pasta imgs e substitua nas variaveis 'users

# Exemplo: 
# user1 = 'imgs/imagemdouser1.png'
# user2 = 'imgs/imagemdouser2.png'
# user3 = 'imgs/imagemdouser3.png'
